type name(parameters)
{
	...;
	group-of-statements;
	...;
}

type - тип возвращаемого значения.
void - пустота.
return - прерывает работу функции, и возвращает значение на место вызова,
		 а так же возвращает управление на место вызова.
name - имя функции.
parameters - принимаемые параметры функции.
group-of-statements - это тело функции.
