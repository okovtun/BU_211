Repository - это хранилище исходных кодов.
Remote repository - удаленный репозиторий (хранящийся на сервере).

NAS - Network Attached Storage

fetch, pull и push
fetch - скачать базу данных репозитория с удаленного репозитория.
pull - скачать и распаковать базу данных с удаоенного репозитория.
push - закачать все контрольные точки с локального репозитория на удаленный репозиторий.

commit - это контрольная точка, в которую сохраняется текущее состояние всех нужных файлов в репозитории.
